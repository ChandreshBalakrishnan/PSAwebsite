Prototype System Overview (Folder 04_Prototype)

This prototype exemplifies a single module of an embedded greenhouse lighting system designed to change light color in response to user input. The system integrates a software control subsystem responsible for managing system behavior (Section 2), an electrical subsystem that interfaces user inputs and LED outputs (Section 3), and a digitally fabricated structural enclosure that houses and protects the system components (Section 4). The prototype was developed in accordance with the defined specifications and design constraints. System performance was evaluated against the specifications using a structured verification framework, as documented in Section 6.
